# NPM Package Vulnerability Scanner

CLI that scans package.json for outdated deps, known vulnerabilities, and license issues.

## Tech Stack

- Node.js
- Python
- Bash

## Getting Started

```bash
# Clone the repo
git clone https://github.com/JASSBR/npm-package-scanner.git
cd npm-package-scanner

# Install dependencies (if any)
npm install

# Run the project
npm start
```
